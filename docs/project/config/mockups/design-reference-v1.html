1. Terminal state taxonomy — 6 colors

Each terminal is in exactly one state at any moment. Project pill aggregates to the most-urgent state across its terminals. done-unack auto-clears the moment the user focuses that tab.

idle
Gray. Shell prompt waiting / no Claude session active.
running
Green. Claude is thinking, or a process is actively executing.
awaiting-input
Blue. Claude returned a question, waiting on user.
needs-permission
Yellow. A tool call requires explicit approval.
error
Red. Process exited non-zero, or Claude reported an error.
done-unack
Purple. Just finished — clears the moment user focuses the tab.

Aggregation priority (project pill picks the highest)

error  >  needs-permission  >  awaiting-input  >  done-unack  >  running  >  idle
